## Por que Next.js?

Next.js é o framework React mais popular para criação de sites modernos. Ele oferece renderização do lado do servidor (SSR), geração de páginas estáticas e uma experiência de desenvolvimento incrível.

### Principais vantagens

- **Performance** — páginas carregam rapidamente com SSR e SSG
- **SEO** — metadados otimizados automaticamente
- **Roteamento** — sistema baseado em arquivos, sem configuração
- **Deploy simples** — integração nativa com a Vercel

## Tailwind CSS

Tailwind mudou a forma como estilizamos sites. Em vez de escrever CSS personalizado, usamos classes utilitárias diretamente no HTML/JSX.

```css
/* Sem Tailwind */
.button {
  background-color: #3b82f6;
  padding: 12px 24px;
  border-radius: 8px;
}
```

```html
<!-- Com Tailwind -->
<button class="bg-blue-500 px-6 py-3 rounded-lg">
  Clique aqui
</button>
```

## Conclusão

Com Next.js + Tailwind, você tem uma base sólida para qualquer projeto web. Comece pequeno e evolua conforme necessário.
